Abstract

Traditional portfolio optimization methods ignore valuable textual information that shapes market narratives. We introduce NarrativeBreak, a novel framework that integrates structural break detection with multi-source NLP signals for dynamic asset allocation. Unlike prior work that treats sentiment as static features, our approach detects narrative regime changes that precede price regime shifts by 5.7 trading days (95% CI: 1.2-10.2). We contribute: (1) a hierarchical sentiment aggregation scheme introducing "narrative beta"; (2) confidence-calibrated Black-Litterman views from ensemble model disagreement; (3) the first systematic comparison of lexicon, transformer, and LLM methods for portfolio performance. On a multi-asset universe spanning 2010-2023, NarrativeBreak achieves a Sharpe ratio improvement of 83% over equal-weight portfolios, with 26% lower maximum drawdown. While these improvements are economically meaningful, statistical significance is marginal after multiple testing correction (Bonferroni-adjusted p = 0.17).

Key Contributions

1. NarrativeBreak Framework

First unified framework integrating structural break detection with NLP signals, detecting narrative regime changes 5.7 days before price-based methods.

2. Hierarchical Sentiment

Three-level aggregation (asset to sector to market) introducing "narrative beta" - measuring sensitivity to market-wide narrative shifts.

3. Confidence Calibration

Principled uncertainty quantification using ensemble disagreement for Black-Litterman integration.

4. NLP Benchmark

First systematic comparison of VADER, FinBERT, and LM Dictionary for portfolio performance on 40,072 samples.
