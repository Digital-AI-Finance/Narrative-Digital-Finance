Research Questions

RQ1: Can a combined ML approach outperform single methods for structural break detection?

RQ2: Do complex ensemble methods outperform simple forecast combinations?

Answer: Yes to both. The stacking ensemble significantly outperforms both single-stream models and simple averaging approaches.

Methodology

Feature Streams

  • Quantitative: Returns, volatility, momentum, Hurst exponent
  • NLP: Sentiment, topic distributions, polarity drift (TOPol-style)
  • Break Signals: PELT, CUSUM, BOCPD, Chow test

Model Architecture

  • Base models: XGBoost classifiers per feature stream
  • Meta-learner: Stacking ensemble with logistic regression
  • Validation: Walk-forward with monthly retraining
