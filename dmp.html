1.2 How will the data be collected, observed or generated?

Data Collection Methods:

  • API Access: FRED API (Python fredapi), SEC EDGAR XBRL API, BIS web scraping (Beautiful Soup)
  • Commercial Pipelines: RavenPack SQL access, LSEG API and bulk delivery, Deutsche Borse secure file transfer
  • NLP Processing: HuggingFace transformers, BERT/FinBERT embeddings, BERTopic modeling

Quality Assurance:

  • Automated validation scripts for data integrity
  • Statistical consistency checks (missing values, outliers, distribution analysis)
  • Cross-validation with alternative data sources where available
  • Version control for all processing code

Versioning:

  • Code: GitHub (Digital-AI-Finance organization)
  • Data: Zenodo integration for DOI assignment on releases
  • Models: MLflow for experiment tracking and model registry

2. Ethics, Legal and Security Issues

2.1 How will ethical issues be addressed and handled?

Ethical Considerations:

  • No Personal Data: All financial data is aggregated market data with no individual identification
  • Public Statements: Central bank speeches are official public communications
  • Responsible AI: Transparency in model documentation, bias assessment for NLP models
  • No Trading: Research outputs are not used for automated trading decisions

Data Provider Agreements:

ProviderLicense TermsRestrictions
Refinitiv/LSEGAcademic research licenseNo redistribution of raw data
RavenPackResearch subscriptionAcademic use only
Deutsche BorseResearch collaboration (Dr. Schlamp)Project-specific usage

Ethics Approval: Not required (non-human-subjects research per Swiss SNSF guidelines)

4.2 Are there any necessary limitations to protect sensitive data?

Unrestricted (Open Access):

  • All code developed by the project (MIT License)
  • Derived features, embeddings, and processed outputs (CC-BY 4.0)
  • Documentation, methodology, and research materials
  • Public source data (FRED, BIS, SEC)

Restricted (Cannot Share):

  • Raw RavenPack news data - license prohibits redistribution
  • Raw LSEG/Refinitiv transcripts - license prohibits redistribution
  • Raw Deutsche Borse T7 data - collaboration agreement restriction

Reuse Conditions:

  • Citation required (BibTeX provided in repositories)
  • License compliance: MIT for code, CC-BY 4.0 for data/models
  • Academic and commercial use permitted for our outputs

Access Requests: For restricted data, contact joerg.osterrieder@bfh.ch - we can facilitate contact with original data providers.
