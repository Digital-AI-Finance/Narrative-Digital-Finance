Detailed Steps

1Clone Repository
git clone https://github.com/Digital-AI-Finance/NarrativeBreak.git cd NarrativeBreak
2Create Virtual Environment
# Linux/Mac python -m venv venv source venv/bin/activate # Windows python -m venv venv .\venv\Scripts\activate
3Install Dependencies
pip install -r requirements.txt
4Run Full Pipeline
python run_full_pipeline.py --seed 42

This runs 6 stages: Experiments, LaTeX Tables, Paper, Charts, Compile, Validate

5Verify Results
python paper/verify_paper_claims.py --strict

Expected: "All 84 paper statistics verified"

Troubleshooting

ModuleNotFoundError: No module named 'experiments' +

Ensure you're running from the project root directory:

cd NarrativeBreak # not a subdirectory
LaTeX compilation fails +

Install full texlive:

# Ubuntu/Debian sudo apt install texlive-full # Windows: Download from tug.org/texlive
Results differ slightly +

Verify seed is set to 42. Floating-point differences < 0.01 are expected across platforms due to different numerical implementations.

Reproducibility Guarantees

  • Deterministic: Same seed (42) produces identical results
  • Platform-independent: Tested on Linux, Mac, Windows
  • Version-pinned: All dependencies locked in requirements.txt
  • Verified: Automated tests check all 84 paper statistics
