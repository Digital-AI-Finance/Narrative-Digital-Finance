Prerequisites

  • Python 3.9+ (tested with 3.10, 3.11)
  • Git for cloning the repository
  • ~500 MB disk space for data and outputs
  • Optional: FRED API key for fresh macroeconomic data

Quick Start (5 minutes)

1
Clone the repository 
git clone https://github.com/Digital-AI-Finance/Narrative-Digital-Finance-CB-Speeches.git
cd Narrative-Digital-Finance-CB-Speeches/cb_speeches
2
Create virtual environment (recommended) 
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python -m venv venv
source venv/bin/activate
3
Install dependencies 
pip install -r requirements.txt
4
Run the full analysis pipeline 
python -c "from analysis.run_all import run_pipeline; run_pipeline(use_cached=True, verbose=True)"
5
Generate all charts 
cd charts
python run_all_charts.py
Done! All outputs are now in the data/ and charts/*/ directories.

Interactive Dashboard

Launch the Streamlit dashboard for interactive exploration:

cd cb_speeches
streamlit run app.py

Opens at http://localhost:8501

Run Tests

Verify data integrity and pipeline outputs:

cd cb_speeches
python -m pytest tests/ -v

Test Coverage

  • test_data_loading.py - Inventory structure, file existence
  • test_data_integrity.py - PCA eigenvalues, sentiment counts, correlations

Fresh Data Download (Optional)

To download fresh macroeconomic data from FRED:

1
Get a free FRED API key

Register at FRED API

2
Set environment variable 
# Windows (PowerShell)
$env:FRED_API_KEY = "your_api_key_here"

# macOS/Linux
export FRED_API_KEY="your_api_key_here"
3
Run pipeline with fresh download 
python -c "from analysis.run_all import run_pipeline; run_pipeline(use_cached=False, verbose=True)"
Note: The repository includes cached macroeconomic data, so a FRED API key is not required for reproduction.

Troubleshooting

ImportError: No module named 'ruptures'

pip install ruptures

FileNotFoundError: speeches parquet

Ensure you're in the cb_speeches/ directory when running commands.

Memory errors on large datasets

The pipeline requires ~2GB RAM. Close other applications if needed.

Chart fonts look different

Charts are designed for PDF output. Screen rendering may vary by system.
