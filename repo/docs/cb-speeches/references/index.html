About OpenAlex Verification

All references are automatically verified using the OpenAlex open academic database. The verification process:

  • Searches OpenAlex by title and author names
  • Verifies DOI, journal, volume, pages, and year
  • Adds missing DOIs and OpenAlex identifiers
  • Auto-corrects minor year discrepancies
  • Reports citation counts from OpenAlex

Run verification locally: cd paper && python verify_references.py
