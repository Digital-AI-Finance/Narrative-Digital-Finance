Data Sources

  • Macroeconomic Data: Federal Reserve Economic Data (FRED) API
  • CB Speeches: BIS/Gigando dataset with NER annotations and pre-computed sentiment labels

Note: Raw speech text is not included due to licensing. Only aggregated monthly sentiment counts are provided.
